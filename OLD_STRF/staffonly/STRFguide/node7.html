Ease of Use

One of the main purposes of software tools is to relieve tedium, not to promote it. Through the use of color, pop-up menu, editable text field, mouse and menu oriented user interface, STRFPAK is designed to keep the user's learning curve easy. It also provides interactive tracing window when the large execution takes place so that the user can easily gain insight into how and where the calculation procedure goes. The dialog windows are also implemented for showing detailed warning, error and confirmation messages.
