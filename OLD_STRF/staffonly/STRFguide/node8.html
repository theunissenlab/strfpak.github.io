Portability

STRFPAK is implemented using Matlab programming language. It can run on any operating systems that support Matlab and the X Window System. Although STRFPAK is effective in color, it also works on monochrome and grayscale monitors. For the future version, we propose to develop standalone executable program that will be independed of Matlab environment.
