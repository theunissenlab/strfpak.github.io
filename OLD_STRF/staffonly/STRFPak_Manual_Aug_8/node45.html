Quick Guide to Run STRFPAK 

Quick Installation about STRF package
---------------------------------
1. Open Matlab first.

2. On the MATLAB command prompt line, type:
   >> strfpak

3. Click each button in order on the main window 
   of STRFPAK to get input data, estimate the STRF, 
   predict a PSTH using the STRF and validate the 
   goodness of fit.

4. All the intermediate results are saved in the output 
   directory you specified under the "Calculate" button 
   on the main window.
