User Manual

This chapter deals with everyday usage of the toolbox. In a step-by-step manner we show how data is loaded, the preprocessing options available, what type of analysis we have, how the intermediate and final results can be saved, and how these results can be viewed. More information on each of the stages, such as a description of the available data formats and the different modeling algorithms, can be found in the later chapters.
