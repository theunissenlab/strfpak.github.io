Online Support and Downloading STRFPAK

The STRFPAK software has its own homepage containing software and documentation. It is updated regularly. 
http://strfpak.berkeley.edu
>From the following website, you can download the Unix/Linux version and Windows version compressed tar files that contain all the Matlab files and sample data sets. 
http://strfpak.berkeley.edu/download.html
The toolbox also has a user mailing list for discussion, problems and announcements. 
strfusers@socrates.berkeley.edu
Since the mailing list is maintained manually, you need send an email to junli@socrates.berkeley.edu to join the list.
