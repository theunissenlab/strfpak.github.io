Portability

STRFPAK is implemented using Matlab programming language. It can run on any operating systems that support Matlab and the X Window System. Although STRFPAK is effective in color, it also works on monochrome and grayscale monitors. For some future version of STRFPAK, we propose to develop a standalone executable program that will be independed of the Matlab environment.
