The STRFPAK Release Notes

This document describes the new features and changes in each release of STRFPAK.

STRFPAK 2.0.1

September 29, 2004.

  • Made it compatible with Matlab 7.0. Thanks to Purvis Bedenbaugh. NOTE: Matlab 7.0 uses different compression and unicode to save data. So beware some STRFPAK results saved using Matlab 7.0 may not be loadable from the previous version of Matlab.
  • Fixed a bug on PSTH estimation in preprocessing stage.
  • Fixed a bug on normalizing information values in validation stage. Thanks to Sarah Woolley.
  • Improved the graphical user interface of STRFPAK mainwindow, spectrogram and scalogram windows.
  • Improved STRFPAK batch-processing template codes' reusability.

STRFPAK 2.0

August 9, 2004.

  • Some speed improvments in calculating auto-correlation and STRF normalization, thanks to Yale E Cohen.
  • Added user-driven preprocessing options.
  • Added new validation method.
  • Added a batch-processing template at request of Shaowen Bao. thanks to Israel Nelken.
  • Added modulation spectra at request of Yale E Cohen and Brian Gygi.
  • Fixed several display scale bugs with help of Lauren Jones.
  • Updated STRFPAK's user manual, thanks to qingbo wang and Tucker Balam.

STRFPAK 1.0

January 03, 2003.

  • Initial release of STRFPAK.
